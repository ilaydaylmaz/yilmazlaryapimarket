# Render.com Manuel Deploy Rehberi

## Deploy Olmadıysa - Manuel Deploy Yapın

### Adım 1: Render.com Dashboard
1. [Render.com](https://render.com) Dashboard'a gidin
2. Service'inize (`yilmazlaryapimarket`) tıklayın

### Adım 2: Manuel Deploy
1. Üst menüden "Manual Deploy" butonuna tıklayın
2. "Deploy latest commit" seçin
3. Deploy başlayacak

### Adım 3: Deploy Durumunu İzleyin
1. "Events" veya "Logs" sekmesine gidin
2. Deploy ilerlemesini görebilirsiniz
3. "Live" yazısı göründüğünde tamamlanmıştır

## Alternatif: GitHub'dan Trigger

Eğer manuel deploy çalışmazsa:
1. GitHub repository'nize gidin
2. Herhangi bir dosyada küçük bir değişiklik yapın (boşluk ekleyin)
3. Commit edin ve push edin
4. Render.com otomatik deploy başlatacak
